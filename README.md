# Features & Functions
>You can setup all values/functions.
### ScreenSpace
- LightShafts (Godrays/Creespecular)
- Bloom
- Vignette
- Tonemapping (Uncharted2TonemapOp/reinhard)
- Crossprocess
- Motion Blur
- RainDrops
- FilmGrain
- Cinematic Border
- RainDesaturation
- Chromation Abberation
- FXAA
- Eye Adaptation
### WorldSpace
- Shadows
- CustomLighting
- AmbientOcclusion (only for testing, by capt tatsu.)
- CustomVanillaSkyRendering
- Custom Atmosphere
- Water
- WavingStuff
- Fog
- Clouds
- Specular Lighting
- Rain
- Fake Shadows
- Fake Caustic
